# Simple ChatAPP With Socket.io
Simple Chatting app i used: 
1. Express
2. socker.io
3. http
## Usage 
```
# clone repo
git clone https://github.com/aladdinalizada/ChatAPP
# install node_modules folder
npm i
# Start app
npm run start
```
